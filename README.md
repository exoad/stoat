# Stoat

**A language focused on dynamic, redundancy-free evaluation primarily designed to act as a versatile, easy-to-use configuration language.**

## "Hello World!" demo

Here is a simple "Hello World!" demo in Stoat.

`main.sto`
```
{
  main
    #__MAIN__():
      ::str hello world;   
}
```
